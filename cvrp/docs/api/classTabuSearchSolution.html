bool IsTabu(const int begin, const int end)¶

Check if set of vector is tabu.

Iterates over vector of vectors to_check to check if any of the moves listed in to_check are tabu. to_check is updated for each move considered

Return
bool True if any of the vectors checked are tabu
Parameters
  • begin: index at which to begin iteration
  • end: index at which to end iteration (inclusive)

bool Aspiration(double cost_increase, double cost_reduction)¶

Aspiration criteria.

Ensures that if a move provides a better solution than the best currently available, the move is accepted even if it is tabu

Return
bool True if the total cost reduction alls the cost of the solution tobe less than the best cost.
Parameters

  • cost_reduction: Reduction in cost due to move
